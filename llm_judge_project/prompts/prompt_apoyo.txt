#   Tu tarea es clasificar cada pregunta en exactamente una de estas dos etiquetas:

#   - `acusatoria`

#   {

#   Objetivo de clasificación:
#   Clasifica por el sentido pragmático global del texto.

#   Marca `acusatoria` cuando la pregunta, de forma explícita o suficientemente imputativa, atribuye, reprocha, denuncia o insinúa que la
#   entidad, sus funcionarios o el proceso actuaron de forma indebida, ilegal, direccionada, parcializada, arbitraria o hecha para favorecer o
#   perjudicar a alguien.

#   También marca `acusatoria` cuando:
#   - se afirma o insinúa de manera suficientemente directa que el proceso está direccionado, redireccionado o hecho para una empresa, marca o
#   proveedor
#   - se atribuye a la entidad una actuación ilegal o contraria a la normativa
#   - se reprocha que los pliegos, requisitos o especificaciones fueron puestos para beneficiar a alguien
#   - se usa sarcasmo, reproche o denuncia que implique actuación indebida del proceso o de la entidad
#   - el texto mezcla pedido de corrección con una imputación suficientemente clara de conducta impropia; en ese caso prevalece `acusatoria`

#   Marca `no_acusatoria` cuando la pregunta sea principalmente:
#   - una solicitud de aclaración
#   - un pedido de ampliación, corrección, equivalencia o cambio de plazo, rango o requisito
#   - una objeción técnica, administrativa o jurídica
#   - una observación sobre participación, concurrencia, marcas o requisitos restrictivos
#   - una cita normativa usada solo para sustentar un pedido de corrección
#   - una crítica al efecto del requisito sin atribuir claramente conducta indebida a la entidad o al proceso

#   Regla clave:
#   - Si el texto describe solo el efecto del requisito o pide que se lo cambie, usa `no_acusatoria`
#   - Si el texto atribuye responsabilidad impropia a la entidad o al proceso por ese efecto, usa `acusatoria`

#   Importante:
#   No uses una regla demasiado estricta. Para esta tarea, también cuenta como `acusatoria` la insinuación o reproche suficientemente claro,
#   aunque no sea una acusación jurídica formal.
